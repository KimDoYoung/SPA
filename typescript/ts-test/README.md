# typescript

## 설치.

1. npm init -y
2. npm install -g typescript
3. tsc -version -4.7.2
4. tsc --init
    - tsconfig.json

## vscode extenstions

-   TypeScript ESLint
-   TypeScript Toolbox
-   TypeScript Importer
-   Ponicode
-   CodeMetrics

-   tslint
    -   npm i -g tslint
    -   tslint --init
-   JavaScript and TypeScript Nightly

## 특징

1. 타입이 존재하는 javascript
2. 런타임이 아닌 컴파일시에 에러를 찾을 수 있다.
3. 클래스, 인터페이스, 제너릭등 자바와 유사하게 OOP개념으로 작성할 수 있다.
