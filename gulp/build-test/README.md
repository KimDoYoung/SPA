gulp-test
=========

## 목적

- gulp 사용법
- gulp 는 grunt와 유사한 task runner 이다.
- gulp version 4부터 이전과 문법이 틀려졌다.

## commands

1. node -v
    - v16.15.0
2. npm -v
    - 8.5.5
3. npm init -y
4. npm install -g gulp-cli
5. npm install -D gulp
6. touch Gulpfile.js
7. import gulp and deconstruct
